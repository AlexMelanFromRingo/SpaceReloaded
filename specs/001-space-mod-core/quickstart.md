# Quickstart: сборка и запуск SpaceReloaded

## Требования

- **JDK 25** (Temurin). Через sdkman: `sdk install java 25.0.3-tem` (в WSL уже установлен: `~/.sdkman/candidates/java/25.0.3-tem`)
- Git; интернет для первого резолва зависимостей
- IntelliJ IDEA **2025.3+** (поддержка Java 25), опционально

## Сборка

```bash
cd ~/SpaceReloaded
JAVA_HOME=~/.sdkman/candidates/java/25.0.3-tem ./gradlew build
```

- `./gradlew :core:test` — юнит-тесты ядра (без Minecraft, быстрые)
- `./gradlew :mod:runClientGametest` — E2E-стенд: реальный клиент + мир, сценарии герметичности/станков/сборки ракеты (~40 с)
- `./gradlew :mod:build` — сборка jar мода (в `mod/build/libs/`)

## Запуск дев-клиента/сервера

```bash
JAVA_HOME=~/.sdkman/candidates/java/25.0.3-tem ./gradlew :mod:runClient
JAVA_HOME=~/.sdkman/candidates/java/25.0.3-tem ./gradlew :mod:runServer
```

## Ручная проверка среза P1+P2 (герметичность, энергия, шлюз)

Все блоки — в креативной вкладке SpaceReloaded.

1. Создать мир, выполнить `/spacereloaded debug vacuum on` (режим вакуума в текущем измерении; душит всех живых существ без маски вне зон, креатив не трогает).
2. Построить куб 5×5×5 из герметичной обшивки/стекла, внутрь — контроллер атмосферы, вплотную (или через энергокабель) — блок «Креативный источник энергии» (либо честно: РИТЭГ/солнечную панель).
3. Убедиться: `/spacereloaded debug zone` изнутри — статус ЗАМКНУТО, атмосфера растёт.
4. Убрать угловой блок, оставив диагональное касание — статус УТЕЧКА (26 направлений!), декомпрессия толкает к пробоине.
5. Шлюз: тамбур с двумя герметичными люками. Открыть второй при открытом первом нельзя (интерлок), открытие идёт через цикл выравнивания с шипением.
6. Кислородная маска на голове (слот шлема) спасает от вакуума снаружи — работает и на мобах.

## Ручная проверка US7 (орбитальная пушка)

1. Целеуказателем (ПКМ) отметить блок на поверхности Земли — метка хранится в самом пульте.
2. Улететь на орбиту, поставить пушку, подключить энергию (кабель от РИТЭГ/батареи, нужно 50к E).
3. ПКМ вольфрамовым ломом — зарядить (до 16), ПКМ целеуказателем — навести по метке, ПКМ пустой рукой — выстрел.
4. Sneak+ПКМ — статус (ломы/энергия/цель). Внизу за ~4 с до удара — гром и огненная колонна над целью.
5. Удар честный: кратер из E = ½mv² (масса и скорость лома — в конфиге), бедрок/вода/обсидиан выносятся НЕ будут.
6. **Пульт**: Sneak+ПКМ целеуказателем по пушке — привязка (пультов может быть много, по одному на пушку).
   После привязки: новая метка ПКМ по блоку сразу перенаводит эту пушку из любого измерения;
   ПКМ в воздух — дистанционный выстрел; Sneak+ПКМ в воздух — статус пушки удалённо.

## Полезное

- Версии тулчейна — в `gradle.properties` (см. `specs/001-space-mod-core/research.md`, R1).
- Исходники Minecraft 26.2 деобфусцированы: `./gradlew :mod:genSources` больше не нужен в старом смысле — Loom подключает исходники напрямую; читать имена API по jar, не по памяти.
- Конфиг мода: `run/config/spacereloaded.json` (лимиты радиуса зон, блоков ракеты, параллельных пересчётов).
