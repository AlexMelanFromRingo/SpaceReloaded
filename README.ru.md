Космос для Minecraft 26.2 с одним правилом: **физика настоящая**. Ракета —
сущность, собранная из блоков, которые вы поставили сами. Запас скорости
считается по формуле Циолковского, тяговооружённость — по реальным массам
деталей, а кривая ракета опрокидывается на взлёте.

[English documentation](README.md)

## Что внутри

- **Ракеты произвольной формы.** Строите что угодно на стартовой площадке,
  ПКМ по пилону — конструкция взлетает единой сущностью. Масса, тяга, центр
  масс и моменты — из реальных блоков. TWR меньше единицы — остаётесь на
  земле. Топливо берётся из тех баков, которые вы залили.
- **Два топлива с разными характерами.** Керолокс (плотный, тяговитый) — из
  нефтеносного сланца через перегонный куб; гидролокс (высокий удельный
  импульс) — электролизом льда. Двигатель жжёт один тип: смешанный стек не
  соберётся.
- **Герметичность, которая уважает геометрию.** Проверка помещения — флудфилл
  в 26 направлениях: диагональная щель в углу — это утечка, ровно как
  недостроенная рамка портала. Шлюзы с интерлоком, пробоина — декомпрессия,
  тянущая к дыре.
- **Орбита Земли и Луна.** Измерения с масштабом координат 1:8, орбитальная
  платформа над точкой старта, ISRU на Луне: лёд → гидролокс + кислород в
  баллоны.
- **Стыковка.** Блок «стыковочный узел» — плоскость разделения: расстыковали
  припаркованный стек на носитель и лендер, слетали вниз, заправились,
  вернулись, состыковались. Захват в радиусе — парковаться пиксель-в-пиксель
  не нужно. Топливо делится и суммируется по реальной ёмкости баков.
- **Беспилотные рейсы.** Полётная программа (цель + посадочный маяк)
  загружается в припаркованную ракету, запуск — дистанционно. Автопилот
  набирает высоту, выполняет переход и садится на маяк тормозным импульсом.
- **Орбитальная кинетическая бомбардировка.** Пушка работает только на
  орбите, стреляет вольфрамовыми ломами по честной траектории входа. Радиус
  кратера — из E = ½mv² с кубическим подобием; обсидиан и вода держат удар.
  Наведение и выстрел — пультом из любого измерения.
- **Энергетика.** Единицы Team Reborn Energy: угольный генератор на старте,
  солнечные панели (×1.5 в вакууме), РИТЭГи для теневой стороны, кабельные
  сети, аккумуляторы.
- **Ведущая нить.** Шестнадцать достижений — от первого стального слитка до
  замкнутой межпланетной петли. Полная хронология — в
  [progression.md](specs/001-space-mod-core/progression.md).

## Никакой телепорт-магии

Ракета не превращается в предмет в инвентаре. Вернуться домой — значит
заправиться через ISRU, состыковаться с носителем или построить титановую
возвратную капсулу, выдерживающую посадку до 25 м/с. Межпространственные
события (кинетические удары, беспилотные рейсы) держат чанки
само-протухающими ticket'ами: полёт завершится даже без игроков рядом и
переживёт перезапуск сервера.

## Сборка из исходников

Нужен JDK 25 (Temurin).

```bash
./gradlew build                     # всё + юнит-тесты
./gradlew :core:test                # тесты физического ядра, без Minecraft
./gradlew :mod:runClientGametest    # E2E-стенд: реальный клиент, 6 сценариев, ~60 с
./gradlew :mod:runClient            # дев-клиент
```

Jar появляется в `mod/build/libs/`.

## Архитектура

- `core/` — чистая Java-физика: флудфилл, калькулятор Циолковского/TWR,
  интегратор полёта с гиро-компенсацией, баллистика. Ни одного импорта
  Minecraft, покрыто юнит-тестами.
- `mod/` — слой Fabric: сущности, станки, измерения, сеть. Детали, топлива и
  планеты — datapack-реестры: аддон может добавить планету или двигатель
  одним JSON.
- Сервер авторитетен везде; пересчёт герметичности — в фоновых потоках по
  палитровым снимкам, снятым в главном потоке.
- E2E-стенд (`mod/src/gametest/`) поднимает настоящий клиент и на каждое
  изменение прогоняет герметичность, промышленность, сборку, межпространственную
  бомбардировку, стыковку и полётные программы.

Проектная документация — в [`specs/001-space-mod-core/`](specs/001-space-mod-core/):
спецификация, план, модель данных, прогрессия, шпаргалка по внутренностям
Minecraft 26.2 и [backlog](specs/001-space-mod-core/inspiration-backlog.md)
механик, адаптированных из Advanced Rocketry и Galacticraft.
